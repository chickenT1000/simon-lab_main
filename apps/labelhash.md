# LabelHash

LabelHash is a Simon-Lab web application for ISO 20560-compliant label generation for chemical process equipment and pipelines.

## Canonical pages
- Detail page: https://simon-lab.org/apps/labelhash.html
- Product URL: https://label.simon-lab.org

## What it does
- Generates process equipment and pipeline labels
- Supports ISO 20560-compliant layouts
- Includes hazard warnings and GHS pictograms
- Applies color-coded media identification
- Provides live preview and print-ready export
