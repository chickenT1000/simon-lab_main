# ProcessView

ProcessView is a Simon-Lab web application for web-based P&ID visualization with embedded physics simulation for chemical plant design and control.

## Canonical page
- Detail page: https://simon-lab.org/apps/process-view.html

## What it does
- Visualizes process and instrumentation diagrams in the browser
- Supports embedded physics simulation
- Helps validate process flow and control logic
- Supports engineering review before construction or commissioning
- Provides technical drawing export workflows
